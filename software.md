## youtube-dl
1. 安装python
2. 安装ffmpeg
3. 下载youtube-dl
4. 运行
```
//下载某个列表
cd d:\youtube
.\youtube-dl.exe -i -f bestvideo+bestaudio --merge-output-format mp4  "https://www.youtube.com/playlist?list=PL8mPWv3h4qJcxyNXAINRXEqHF06cc1Euq"
```
